
# ProyectoM1_JorgeMora

Se presenta este documento con la descripción de la tarea a realizar, que es una página web  estática e interactiva para el cliente “Colorfly Studio”, quien solicita una web para generar paletas de color aleatorias

Descripción del proyecto.

- Botón generador de paletas de colores.
- Número de opciones de color: 6, 8 o 9.
- Generar colores aleatorios en dos formatos: HSL y HEX
- Mostrar la tarjeta de color junto a su código de color.
